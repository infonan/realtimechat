# 💬 Real-Time Chat App

A real-time chat application built with Node.js, Express, and Socket.io.

## 🚀 Features

- Login with username
- Real-time messaging
- Message history (last 50 messages)
- Join/leave notifications
- Dark/light mode toggle
- Stay logged in after refresh
- Logout option
